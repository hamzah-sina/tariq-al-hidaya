# طريق الهداية (Tariq Al-Hidaya)

تطبيق Flutter للصحة الشخصية والتعافي من الإدمان بمنهج إسلامي روحي، يجمع بين تتبّع الانتكاسات، بناء العادات الإيجابية، ومكتبة دينية مرجعية — كل ذلك بتخزين محلي بالكامل وبدون أي اعتماد على سيرفر خارجي.

## الفكرة

التطبيق لا يركّز فقط على "لا تفعل X" — بل يوازن بين تتبّع التعافي من الإدمان وتتبّع بناء عادات إيجابية موازية (الصلاة، القرآن، الذكر)، مع تصميم نفسي متعمّد يتجنّب الشعور بالعار عند الانتكاس (Abstinence Violation Effect) عبر آليات مثل تجميد السلسلة والعرض متعدد الطبقات.

## الميزات

### تتبّع التعافي
- عداد أيام تلقائي محسوب من تاريخ آخر انتكاسة (لا يتوقف لو نسيت فتح التطبيق)
- تسجيل انتكاسة مع تصنيف السبب (محفّز) وملاحظة اختيارية
- سجل كامل للانتكاسات مع التاريخ والمحفّز

### العادات الإيجابية
- تتبّع الصلوات الخمس يومياً
- **وِرد قرآن بمستويين:** حد أدنى (لا يُكسر، قابل للتخصيص) + هدف موسّع اختياري، بعدّاد صفحات فعلي
- تتبّع الذكر اليومي
- **سلسلة لكل عادة على حدة** (Streak per habit) — لا مؤشر واحد عام
- **تجميد السلسلة:** رصيد واحد أسبوعياً يحمي من انكسار السلسلة البصري عند فوات يوم، بدون التظاهر بإتمام العادة فعلياً
- **عرض متعدد الطبقات:** كل عادة تُعرض بثلاث قيم معاً — السلسلة الحالية، أطول سلسلة، نسبة إنجاز الشهر — لمنع فوات يوم واحد من تصفير كل مؤشرات التحفيز

### المحتوى الديني
- **صحيح مسلم كامل** (٧٥٦٣ حديثاً بالإسناد) — تصفّح بالكتب (٥٧ كتاباً) + بحث نصي كامل، من مصدر مفتوح موثّق
- مكتبة كتب ومقالات للتعافي، مع حديث مصاحب لكل عنصر
- اقتباسات قرآنية وتحفيزية يومية

### أدوات فورية (للحظات الإغراء)
- زر طوارئ بتمرين تنفّس موجّه (٤-٤-٤، خمس دورات)
- ذكر فوري بعدّاد تكرار
- أنشطة بديلة مقترحة (تمرين، قراءة، مشي)

### التخصيص
- "عهدي الشخصي": تذكير شخصي يومي بصياغة واضحة — **ليس نذراً شرعياً**
- ٥ أنظمة ألوان (أخضر إسلامي، أزرق الليل، بني المخطوطات، رمادي عصري، أخضر فاتح)
- إشعار يومي بوقت قابل للتخصيص

### الإحصائيات
- تقويم آخر ٣٠/٣٥ يوماً
- تحليل أسباب الانتكاسات (نسب لكل محفّز)

## التقنية

| | |
|---|---|
| **إطار العمل** | Flutter (Dart) |
| **إدارة الحالة** | Riverpod (AsyncNotifier) |
| **التخزين** | shared_preferences (محلي بالكامل، صفر سيرفر) |
| **الإشعارات** | flutter_local_notifications |
| **فتح الروابط** | url_launcher |
| **بناء APK** | Codemagic CI (macOS M2 instance) |

### إصدارات البناء المستقرة (مهم!)
بعد رحلة طويلة من التصحيح، هذه التركيبة هي المستقرة فعلياً مع Codemagic:

```
Gradle:  8.10.2   (مُثبَّت في codemagic بيئة العمل — لا يتجاوز 8.7 فعلياً بالواقع)
AGP:     8.6.0
Kotlin:  2.0.21
Flutter: 3.27.4   (مُحدَّد صراحة في Codemagic UI، لا "channel: stable")
```

⚠️ **تنبيهات حرجة لمن يعدّل ملفات Gradle:**
- صيغة `coreLibraryDesugaring` يجب أن تكون **3 أجزاء**: `"com.android.tools:desugar_jdk_libs:VERSION"` (نقطتان، لا نقطة) — خطأ شائع تسبّب بساعات تصحيح
- `androidx.window` و `androidx.datastore` (يدخل عبر `flutter_local_notifications`/`shared_preferences`) قد يحملان AAR metadata بـ variant غير معروف لـ AGP 8.6.0 — محلول حالياً بفرض إصدارات أقدم (`androidx.window:1.0.0`, `androidx.datastore:*:1.1.1`) في `android/build.gradle`
- المشروع يستخدم **Codemagic UI workflow**، لا `codemagic.yaml` — أي تعديل على ملف YAML في الريبو **لن يُقرأ** إطلاقاً

## مصدر صحيح مسلم

نص الحديث مأخوذ من مستودع مفتوح موثّق: [`fawazahmed0/hadith-api`](https://github.com/fawazahmed0/hadith-api) — تم التحقق من دقة نموذج من النص قبل الإدراج. النص الديني نفسه مصدره القرن التاسع الهجري، عام (Public Domain)، والمستودع مجرد تجميع بصيغة JSON.

## الخصوصية

كل البيانات (الانتكاسات، العادات، العهد الشخصي) تُخزَّن **محلياً على الجهاز فقط** عبر `shared_preferences`. لا حسابات، لا تسجيل دخول، لا سيرفر، لا مشاركة بيانات مع أي طرف ثالث. هذا قرار تصميمي متعمّد نظراً لحساسية بيانات التعافي من الإدمان.

## ميزات مؤجّلة (تحتاج كود Android أصلي)

- **ربط الإشعارات بأوقات الصلاة الفعلية** (بدل وقت ثابت) — يحتاج محرك حساب فلكي
- **وضع تركيز/حظر تطبيقات** أثناء الورد — يحتاج `UsageStatsManager`/`AccessibilityService` عبر MethodChannel مخصّص، مع قيود نشر على Google Play تستوجب Policy Justification

## سير العمل

المشروع يُطوَّر بالكامل من الهاتف (Termux + مدير ملفات)، بدون حاسوب:
1. تعديل الكود محلياً → `git push`
2. Codemagic يبني APK تلقائياً عبر GitHub webhook
3. تحميل APK من Codemagic أو نقله مباشرة
